install a requirements.txt in windows10 without failing
* Open conda terminal into the specified environment (PIP should be already installed)
> FOR /F %p IN (requirements.txt) DO pip install %p
> conda list

Convert tif images to png

- Install imageMagik
- In Windows powerShell
> cd ....\DRIVE\training\images
> magick mogrify -format png -path ../imagesPNG *.tif
